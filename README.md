# Installation
- `clone` this repository
1. Backend
    - `cd backend`
    - `npm install` to install all dependancies
    - start server by  `npm start`

2. Frontend
    - `cd autofill`
    - `npm install` to install all dependancies
    - `npm run dev` to start frontend server
